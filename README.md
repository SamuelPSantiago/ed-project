## Projeto de Estrutura de Dados em C

Este repositório reúne uma série de projetos desenvolvidos na linguagem C com o objetivo de aplicar, de forma prática e didática, os principais conceitos da disciplina de Estrutura de Dados.

Cada projeto foi construído no formato de jogo em modo CLI, permitindo a visualização clara do funcionamento das estruturas de dados por meio de situações reais e interativas.
